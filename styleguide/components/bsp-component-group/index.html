The bsp-component-group component is used when we need to have a site specific group of components that acts as a single component. What that means is that they will all be related to the same data source.

As an example, you might want a group of image promos. You want the first one to be a large single image, and then you want 2 image promos next to each. Rather than trying to create an image promo "option" for that group/combination, you would use this component and then use your JSON to specify what components you want in what order
